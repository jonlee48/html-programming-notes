Lesson 1: The Basics of the Web and HTML

How the Web Works

The World Wide Web (WWW) is a bunch of computers that communicate with each other. When a person goes to a web page like www.google.com, their computer sends a HTTP (Hypertext Transfer Protocol) Request to a server. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser (ex. Safari) interprets the page and displays it on the user's screen.

HTML

HTML stands for Hypertext Markup Language. It is a programming language that forms the majority of the content of the web. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". This video gives a good overview. Here is how HTML documents are structured.

Lesson 2: Creating a Structured Document with HTML

The Box Structure of HTML

In a HTML document, each HTML tag creates an element in the DOM (Document Object Model) that the browser uses to display the page. The elements are organized in a rectangular tree structure. The "tree-like structure" in HTML code comes from the fact that HTML elements can have other elements inside of them. In a browser, this structure shows up as a series of nested boxes. There are boxes inside of boxes inside of boxes, and so on.

Lesson 3: Adding CSS Style to HTML Structure

Avoiding Repetition

Avoiding repetition is is important for a variety of reasons...

Errors: Doing the same thing over and over everytime a code appears is tedious. This can lead to errors when the programmer decides to make a change to something. If they don't dilligently make that same change everywhere the repeated code appears, problems will arise.

Consistency: To make the webpage look nicer I had to be consistent with the styles. Instead of manually giving each code the same style, I was able to assign each lesson div the same class and then specify the style for that class in only one place! By avoiding repetion, it's now very easy for me to make changes to this style.

CSS

CSS stands for (Cascading Style Sheets) and is a coding language that controls the style of HTML elements. Programmers add CSS to change the looks of their webpage. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class. This is a great video on how CSS works.
